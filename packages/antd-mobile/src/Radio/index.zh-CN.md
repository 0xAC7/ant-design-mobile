### API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| checked | 受控组件是否选中 | boolean | - |
| defaultChecked | 非受控组件是否选中 | boolean | false |
| disabled | 禁用 | boolean | false |
| onChange | 变化回调 | (e: React.ChangeEvent\<HTMLInputElement\>) => void | - |
| id | 单选框 id，和 label 配合扩大点击范围 | string | - |
| name | 单选框 name | string | - |
| value | 单选框 value | any | - |
