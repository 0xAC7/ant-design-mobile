## 如何使用

上下留白,设计上通常需要一段特定高度的空白区块

## API

| 成员        | 说明           | 类型      |  可选值         | 默认值       |
|------------|----------------|--------------------|--------------|
| mode       |  上下留白的间距,单位是像素   | number | `4`、`8`、`12`、`16`、`20`、`24`、`28`、`32`、`36`、`40`、`44`、`48` | `8`  |
