## API

### TabItem
| 成员        | 说明           | 类型               | 默认值       |
|------------|----------------|--------------------|--------------|
| value    |    搜索框的默认值     | String |    |
| placeholder    |    placeholder     | String |    |
| onSubmit    |    点击键盘的enter会触发submit事件     | Func |    |
| onChange    |    value变化     | Func |    |
| onClear    |    点击clear图标     | Func |    |
| onCancel    |    点击取消     | Func |    ||
