## API


| 成员        | 说明           | 类型               | 默认值       |
|------------|----------------|--------------------|--------------|
| mode    | 按钮类型,`blue`、`white` 、 `red` 、`warn` 、`light`      | String |   `blue`  |
| size       | 按钮大小，`large`、`middle`、`small`、`little`、`tiny`        | String |   `large`  |
| inline       | 是否是行内按钮        | boolean |  false  |
| disabled   | 是否不可用 | boolean |   false  |
| onClick     | 点击按钮的回调函数 | Function |   无  |
