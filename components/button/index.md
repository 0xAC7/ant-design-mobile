## API

| 成员        | 说明           | 类型      |   可选值      | 默认值       |
|------------|----------------|--------------------|--------------|
| type       | 按钮类型        |   string    | `primary` |   -  |
| size       | 按钮大小        | string   | `large`、`middle`、`small`、`little`、`tiny` |   `large`  |
| ghost     | 是否是幽灵按钮  | boolean |  |  false  |
| inline     | 是否是行内按钮   | boolean |  |  false  |
| disabled   | 是否不可用      | boolean |  |   false  |
| onClick    | 点击按钮的回调函数 | Function|  |   无  |
