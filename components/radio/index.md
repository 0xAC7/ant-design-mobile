## 何时使用

Checkbox当type为`normal`时,props会传给List.Item,thumb指定为一个checkbox

## API


| 成员        | 说明           | 类型       |  可选值        | 默认值       |
|------------|----------------|----------|----------|--------------|
| prefixCls    |         | Object |   | 无  |
| type    | 复选框类型 | String |  `normal`、 `agree` |   'normal'  |
| style    |         | Object |   | 无  |
| name    |         | String |   | 无  |
| checked    |        | String |   | 无  |
| disabled      |         | String | |  无  |
| onChange    | change事件触发的回调函数,参数是event对象 | Function | |   无  |
