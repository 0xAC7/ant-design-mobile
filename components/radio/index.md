## 何时使用

Radio提供了一个简单的方法来从一个预定义集合中选取单个值。

## API

### Radio

| 成员        | 说明           | 类型       |  可选值        | 默认值       |
|------------|----------------|----------|----------|--------------|
| type    | 复选框类型 | String |  `normal`、 `agree` |   'normal'  |
| name    |         | String |   | 无  |
| checked    |        | String |   | 无  |
| disabled      |         | String | |  无  |
| onChange    | change事件触发的回调函数,参数是event对象 | Function | |   无  |
