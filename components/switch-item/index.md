## API


| 成员        | 说明           | 类型               | 默认值       |
|------------|----------------|--------------------|--------------|
| label      | 文案说明        | String           |   无  |
| name       | switch的name    | String           |   无  |
| checked    | 是否默认选中    | Boolean           |   false  |
| disabled   | 是否不可修改    | Boolean           |   false  |
| onChange   | switch change事件触发的回调函数,参数是checked的值 | Function |   无  |
