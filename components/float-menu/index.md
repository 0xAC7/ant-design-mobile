## 如何使用
提示层式的浮动菜单


## API

| 成员        | 说明           | 类型         | 默认值       |
|------------|----------------|-------------|--------------|
| visible   | 当前显隐状态    | Boolean |  false   |
| onVisibleChange   | 当显隐状态变化时调用    | Function |  -   |
