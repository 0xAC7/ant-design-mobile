## API

### TabItem
| 成员        | 说明           | 类型               | 默认值       |
|------------|----------------|--------------------|--------------|
| imgUrl    | 插图URL        | string |    |
| title    |    title文案     | string |    |
| brief    |    brief文案     | string |    |
| buttonTxt    |    按钮文案     | string |    |
| buttonMode    |    按钮类型,`blue`、`white` 、`red` 、`warn` 、`light`     | string |  `white`  |
| buttonClick    |    按钮回调函数     | Func |    ||
