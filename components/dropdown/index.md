## 如何使用
顶部下拉菜单


## API

| 成员        | 说明           | 类型         | 默认值       |
|------------|----------------|-------------|--------------|
| visible   | 当前显隐状态    | Boolean |  false   |
| closable   | 是否允许点击遮罩层关闭    | Boolean |  true   |
| onClose   | 关闭时调用    | function |  -   |
